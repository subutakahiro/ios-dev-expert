# SwiftLint compliance and quality control

- プロジェクトの品質とセキュリティを担保するため、[swiftlint.yml](./scripts/.swiftlint.yml) を参照してください。
- プロジェクト生成時にプロジェクトのルートに[swiftlint.yml](./scripts/.swiftlint.yml)を配置してください。
- ファイル生成・編集後にプロジェクトのルートディレクトリで `swiftLint` を実行してください。
- `swiftLint` を実行を実行した結果、修正を行うべき点が生じた場合、ファイルの修正を行なってください。
